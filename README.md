## 环境搭建（windows版）
```
整体主要是根据flutter官网搭建
1.下载flutter sdk： git clone -b stable https://github.com/flutter/flutter.git
2.安装andriod studio编辑器：https://developer.android.google.cn/studio
    安装的过程中会安装andriod sdk
    安装andriod adt（设备模拟器）
    安装flutter插件，顺便安装dart
    安装汉化包：http://www.smzy.com/smzy/down418870.html#a4
3.由于andriod studio自带的adt太耗内存，可安装夜神模拟器：https://www.yeshen.com/
    连接夜神模拟器与andriod studio
    如先每次连接麻烦，可自己配置环境变量，写脚本
    如需要也可将模拟器置顶显示
```
